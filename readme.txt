# Frontend Task ‚Äì Dashboard Project

 A small frontend project that simulates a freelancing request.  
The project demonstrates using HTML, CSS, Bootstrap, Animate.css, JavaScript, jQuery, DataTables, JSONPlaceholder API, and Toastr.

## ‚ú® Features

- **Loader (Spinner)** on page load with Animate.css fade-out.
- **Responsive Navbar** (Bootstrap).
- **Dark / Light Mode Toggle** (saved in LocalStorage).
- **Dashboard Page**:
  - Statistics cards (Users / Posts / Comments) from API.
- **Users Page**:
  - DataTables integration with pagination, search, and sorting.
  - View user details in a table.
  - Edit / Delete user (locally).
  - ‚≠ê Mark users as favorite (saved in LocalStorage).
  - Toastr notifications for actions.
- **Posts Page**:
  - Display posts in table/cards from API.
  - Live search by title/content.
  - Add new post locally.
  - Edit / Delete post locally.
  - Show comments for each post (fetched dynamically).
- **Comments Page**:
  - Display comments from API in a structured table.
- **Toastr Notifications** for feedback on actions.
- **LocalStorage** used for:
  - Dark/Light theme.
  - Favorites (users).
